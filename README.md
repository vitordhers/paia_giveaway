## Installation

PAIA GIVE AWAY requires [Rust](https://www.rust-lang.org/) v1.63+ to run.

Install the Rust, Cargo and build in order for it to run

```sh
git init
git pull git@github.com:vitordhers/paia_giveaway.git
cargo build
cargo run
```
